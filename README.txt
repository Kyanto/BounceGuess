BounceGuess

https://kyanto.github.io/BounceGuess/

Le but du jeu est de guess les bounces

Espace pour lancer une partie
Quand la balle se fige, vous devez cliquer sur les murs pour placer des guess. (Clique droit supprime le dernier guess placé, recliquer sur un même mur déplace votre dernier guess)
Vous pouvez relancer la balle après avoir placé autant de guess que vous voulez, mais au premier guess faux, vous retomberez à 0 points.
Plus votre guess est proche du point d'impact, plus il vous rapporte de points.
Puis rappuyez sur Espace pour relancer !


Mode multi-joueur temporaire :
Appuyez sur Tab pour afficher le Seeder. Rentrez la même seed que vos amis, appuyez sur le bouton "Set seed". Les 1000 prochaines parties seront identiques.
